# Eight-Queens-Puzzle

## 8クイーン（Eight-Queens-Puzzle）とは

8クイーン（Eight Queens Puzzle：エイト・クイーン）とは、チェス盤と駒のクイーンを使ったパズルのことを言います。

クイーンは次のように縦横斜めを一直線に進むことができる駒です。

|  x  |  -  |  -  |  x  |  -  |  -  |  x  |  -  |
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
|  -  |  x  |  -  |  x  |  -  |  x  |  -  |  -  |
|  -  |  -  |  x  |  x  |  x  |  -  |  -  |  -  |
|  x  |  x  |  x  |  Q  |  x  |  x  |  x  |  x  |
|  -  |  -  |  x  |  x  |  x  |  -  |  -  |  -  |
|  -  |  x  |  -  |  x  |  -  |  x  |  -  |  -  |
|  X  |  -  |  -  |  x  |  -  |  -  |  x  |  -  |
|  -  |  -  |  -  |  x  |  -  |  -  |  -  |  x  |

クイーンを８つ配置し、互いが利いていない位置に配置するにはどうするかを考える問題。
